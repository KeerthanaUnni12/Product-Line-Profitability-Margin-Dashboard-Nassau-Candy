# 📊 Product Line Profitability & Margin Dashboard

An end-to-end interactive business intelligence dashboard built using Python, Streamlit, MySQL, Pandas, and Plotly to analyze revenue, profit, and margin performance across product lines and divisions.

---

## 🚀 Features

- Dynamic filtering by Division and Product  
- KPI cards showing Total Revenue, Total Profit, and Profit Margin  
- Division-wise performance overview  
- Product profitability leaderboard  
- Pareto Analysis (80/20 rule)  
- Cost vs Sales correlation analysis  
- Interactive visualizations using Plotly  

---

## 🛠 Tech Stack

- Python  
- Streamlit  
- MySQL  
- Pandas  
- Plotly
